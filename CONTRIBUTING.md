# Panduan Kontribusi 🤝

Terima kasih telah tertarik untuk berkontribusi di **FIF Adventure**! Kami sangat menghargai bantuan Anda untuk membuat platform ini menjadi lebih baik.

## Cara Berkontribusi

### 1. Melaporkan Bug 🐛
- Pastikan bug tersebut belum dilaporkan di bagian **Issues**.
- Gunakan template Issue yang disediakan (jika ada).
- Sertakan langkah-langkah untuk mereproduksi bug tersebut.

### 2. Mengajukan Fitur Baru ✨
- Buka Issue baru untuk mendiskusikan fitur yang ingin Anda tambahkan sebelum mulai mengerjakannya.

### 3. Mengirimkan Pull Request (PR) 🚀
- **Fork** repository ini.
- Buat branch baru dari `master` (misal: `feat/nama-fitur` atau `fix/nama-bug`).
- Pastikan kode Anda mengikuti standar proyek.
- Lakukan `npm run build` secara lokal untuk memastikan tidak ada error sebelum push.
- Kirimkan Pull Request ke branch `master`.

## Alur Kerja (Workflow)
Proyek ini menggunakan **CI/CD Pipeline**. PR Anda akan melewati pemeriksaan otomatis:
1. **Build Check**: Memastikan aplikasi dapat dikompilasi tanpa error.
2. **Review**: Tim akan meninjau kode Anda sebelum digabungkan.
